## 許容電流
負荷に電動機がある場合、始動電流が多く必要となるため、許容電流を大きくとる必要がある。
過電流遮断機の定格電流を通常負荷と同様に設計すると、電動機の始動電流により過電流遮断機が動作する(電動機が運転できない)。

IM : 電動機の定格電流
IH : 電動機を除く定格電流
IW : 幹線の許容電流

IM <= IH の場合
IW >= IM + IH

IM > IH の場合
IM <= 50A なら、IW >= 1.25IM + IH
IM >  50A なら、IW >= 1.1IM + IH

配線遮断器の定格電流
3IM + IH
